# nirbav-sankar.github.io

Personal portfolio. Case-file aesthetic. Three files, zero dependencies.

## Stack

Plain HTML, CSS, vanilla JS. Web Audio API for the music. Google Fonts (Fraunces, Public Sans, IBM Plex Mono). No build step.

## Structure

```
├── index.html
├── style.css
└── nirbav.jpg
```

## Local

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Deploy

Cloudflare Pages, Vercel, Netlify, or GitHub Pages — drag the folder in.
